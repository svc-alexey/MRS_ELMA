# User Rules

## Project Context

- Проект — расширение конфигурации 1С ERP, имя расширения `MRS_ОбменЕлма`, префикс новых объектов `MRS_`.
- По умолчанию изменения выполнять в расширении; основную конфигурацию ERP считать внешней/типовой и не менять без явного запроса.
- В репозитории хранится частичная выгрузка метаданных: базовая ERP большая и присутствует не полностью. Отсутствие объекта или модуля в дереве проекта не доказывает его отсутствие в конфигурации.
- При анализе влияния учитывать доработки, модуль общепит и модуль КТ-2000.
- Для 1С-справки, шаблонов, БСП и синтаксиса можно использовать локальные Docker MCP: `1c_templates_mcp` (`8004`), `1c_ssl_mcp` (`8008`), `1c_help_mcp` (`8003`), `1c_syntaxcheck_mcp` (`8002`).
