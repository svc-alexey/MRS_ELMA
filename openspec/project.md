# Проект: MRS обмен елма

> Этот файл сгенерирован автоматически установщиком `1c-rules` на основе
> данных репозитория (`Configuration.xml`, БСП-модуль, состав каталогов
> метаданных). Обновляется при `update` параллельно с остальными правилами.
> Если вы редактируете файл вручную, ваши правки сохраняются — установщик
> отметит его как `userModified` в `.ai-rules.json` и больше не будет
> перезаписывать. Чтобы пересобрать с нуля, удалите файл и запустите `update`.

## Конфигурация

- Имя метаданных: `MRS_ОбменЕлма`
- Синоним: MRS обмен елма
- Тип: расширение конфигурации (CFE)
- Префикс расширения (NamePrefix): `MRS_`

## Платформа

- Совместимость: 1С:Предприятие 8.3.24
- Режим запуска по умолчанию: `ManagedApplication`
- Режим форм: управляемые
- Вариант встроенного языка: Russian

## Стандартная библиотека (БСП)

- Используется: нет (общий модуль `СтандартныеПодсистемыСервер` не обнаружен)

- Справочники: 18

## Соглашения и ограничения

- Язык платформы: 1С (BSL); комментарии и UI-строки — на русском
- Стандарты ИТС, расширенные правилами проекта (см. `AGENTS.md` и каталог on-demand правил активного инструмента)
- Запрет на тернарный оператор `?(...)`, `Сообщить()`, обращение к реквизитам через точку
- Перед написанием кода — поиск по `templatesearch` / `codesearch` / `search_code`
- После написания кода — `syntaxcheck` → `check_1c_code` → `review_1c_code` (≤ 3 раза за цикл)
- Полный список запретов и стандартов — `AGENTS.md`, раздел *Forbidden Calls and Constructs*
